```markdown
1. P(Perceptron)
> 초기 인공 신경망

2. FF(Feed Forward)
> 함수 근사, 가장 간단한 인공 신경망

3. DFF(Deep Feed Forward)
> FF의 확장

4. RBF(Radial Basis Network)
> 함수 근사, 시계열 예측, 분류 및 시스템 제어
```

```markdown
5. RNN(Recurrent Neural Network)
> 필기 인식, 음성 인식

6. LSTM(Long/Short Term Memory)
> 필기 인식, 음성 인식, 번역, 로봇 제어, 게임, 의료 

7. GRU(Gated Recurrent Unit)
> LSTM의 간소화(소형 데이터에서 유리)

13. HN(Hopfield Network)
> 콘텐츠 주소 지정 연관 메모리 사용
```

```markdown
8. AE(Auto Encoder)
> 얼굴 인식, 특징 탐지, 단어 의미 획득

9. VAE(Variational Auto Encoder)
> AE의 수학적 공식 강화

10. DAE(Denoising Auto Encoder)
> 노이즈 복구

11. SAE(Sparse Auto Encoder)
> 분류 성능 향상
```

```markdown
12. MC(Markov Chain)
> 확률적 시뮬레이션, 정보 처리, 음성 처리

14. BM(Boltzmann Machine)
> 이미지 처리, 중급 작업 모델링

15. RBM(Restricted Boltzmann Machine)
> 경사 기반 대조 발산 알고리즘에 효율적

16. DBN(Deep Belief Network)
> 그리디 알고리즘에 효율적
```

```markdown
17. DCN(Deep Convolutional Network)
> 이미지 분석에 효과적

18. DN(Deconvolutional Network)
> CNN에서 정보손실 최소화

19. DCIGN(Deep Convolutional Inverse Graphics Network)
> 3D 렌더링 엔진 학습에 효율적
```
