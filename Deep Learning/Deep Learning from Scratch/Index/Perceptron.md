# 퍼셉트론

퍼셉트론은 1957년에 *프랑크 로젠블라트*가 고안한 알고리즘이다.

퍼셉트론을 통해 신경망 알고리즘이 탄생했기 때문에 퍼셉트론 구조를 배우는 것은 신경망과 딥러닝의 기초가 된다.
